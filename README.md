# ✨ BACK Vettime ✨ #
## This is the steps to launch the server ##

#### Clone the project ####
```sh
    git clone https://github.com/LapertJoseph/BackVetime.git
```

## Installation ##
#### Install dependencies ####
BackVetime requires [Node.js](https://nodejs.org/) v14+ to run. 

```sh
    cd BackeVetime
    npm install
```

#### Launch DB + stored procedures ####
```sh
    npm run schema
    npm run procedure
``` 

#### Start the server ####
```sh
    npm run dev
```
